# Academic Staff Analysis

## Project Overview

This project focuses on the analysis and visualization of academic staff data using Microsoft Excel and R Programming Language.

The study was developed as part of a Data Visualization course and aims to transform raw academic staff data into meaningful insights through statistical analysis and visual representations.

## Technologies Used

- Microsoft Excel
- R Programming Language
- Data Visualization
- Statistical Analysis

## Project Objectives

- Analyze academic staff data
- Perform statistical evaluations
- Create meaningful visualizations
- Identify trends and patterns within the dataset
- Present findings through professional reports and posters

## Project Outputs

- Statistical reports
- Data visualizations
- Academic poster presentation
- Analytical interpretations

- Data Cleaning
- Data Visualization
- Statistical Thinking
